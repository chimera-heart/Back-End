## Path variable configuration
```shell
#这是gradle的命令环境变量
GRADLE_HOME=../gradle
PATH=$PATH:GRADLE_HOME/bin

#这是gradle的本地仓库配置的环境变量
GRADLE_USER_HOME=../
```
